# 🎯 标签（class）解释

该数据集中的样本属于意大利同一地区的三种不同的葡萄栽培者（cultivator）所生产的葡萄酒。每种酒的样本被标记为：

类别编号 类别名称 含义
0 class_0 第一种葡萄酒（59个样本）
1 class_1 第二种葡萄酒（71个样本）
2 class_2 第三种葡萄酒（48个样本）

🔍 注意：虽然具体 cultivator 的名字未公开披露，但每一类对应一个 cultivator（酒庄），即 class_0 表示第一个酒庄的葡萄酒，依此类推。

# 🧠 应用方向

  • 分类任务：如使用 SVM、逻辑回归、神经网络等算法。
